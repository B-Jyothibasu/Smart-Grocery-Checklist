# 🛒 Smart Grocery Checklist

A responsive and user-friendly grocery checklist web application built using **HTML, CSS, and JavaScript**.

This project helps users organize their grocery shopping by selecting items from different categories, tracking purchased items, and saving progress using Local Storage.


### Desktop

1. Open the live website.
2. Click the **Install App** icon in the browser address bar.
3. Click **Install**.

### Android

1. Open the website in **Google Chrome**.
2. Tap the **⋮** menu.
3. Select **Add to Home Screen** or **Install App**.
4. The app will be installed and work like a native mobile application.


## ✨ Features

- 🥛 Category-based grocery organization
- 🔍 Search categories and items
- ➕ Add and remove grocery items
- 🛒 Shopping list
- ✅ Mark items as purchased
- 📊 Progress tracker
- 💾 Local Storage support
- 📱 Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage
